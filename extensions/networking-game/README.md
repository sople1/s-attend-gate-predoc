# 네트워킹 게임

참가자 간 상호작용을 증진시키는 네트워킹 게임 확장 기능입니다.

## 기능

- 참가자 매칭
- 실시간 채팅
- 미션 시스템
- 업적 및 보상

## 기술 스택

- Mobile App: Flutter
- Backend: Phoenix (Elixir)
- Database: PostgreSQL
- Real-time: Phoenix Channels
