# 게이트 운영자 행동

게이트에서 실제 출입을 관리하는 운영자의 행동 패턴입니다.

## 시나리오 개요
게이트 운영자가 참가자의 출입을 관리하고 문제 상황에 대응하는 과정을 설명합니다.

## 사용자 프로필
- Tom (28세, 게이트 운영자)
- 현장 경험 풍부
- 대인 커뮤니케이션 능숙
- 빠른 판단력

## 행동 패턴

사용자 행동:
1. 게이트 상태 확인
2. 참가자 접근 모니터링
3. 자동/수동 출입 관리
4. 예외 상황 처리
5. 실시간 보고

감정 변화:
- 시작: 경계, 집중
- 진행: 침착, 주의깊음
- 완료: 만족감, 책임감

## 성공 지표
- 원활한 출입 관리
- 최소 대기 시간
- 문제 상황 해결

## 제약사항
- 장시간 근무
- 혼잡 상황
- 의사소통 필요
