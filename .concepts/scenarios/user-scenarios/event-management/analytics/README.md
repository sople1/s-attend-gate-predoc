# 분석 시나리오

## 개요

Event Management 시스템의 분석 및 리포팅 관련 사용자 시나리오를 포함합니다.

## 구성

### 실시간 분석
1. [실시간 성과 분석](./realtime-performance.md)

   - 트렌드 분석
   - AI 인사이트

2. [데이터 시각화](./visualization.md)
   - 실시간 대시보드
   - 인터랙티브 차트

### 리포팅
3. [성과 리포팅](./reporting.md)
   - KPI 리포트
   - ROI 분석
   - 개선점 도출

4. [AI 분석](./ai-insights.md)
   - 예측 분석
   - 패턴 발견
   - 추천 생성

1. 행사 기획자

2. 데이터 분석가
   - 심층 분석
