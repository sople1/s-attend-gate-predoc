# 시스템 흐름 / 데이터 처리

이 디렉토리는 s-attend-gate 시스템의 데이터 처리 관련 시스템 흐름을 정의하는 스토리보드들을 포함합니다.

## 포함되는 스토리보드

- 이벤트 데이터 처리 흐름
- 사용자 데이터 동기화 흐름
- 통계 데이터 처리 흐름
- 로그 데이터 처리 흐름

## 구조 규칙

- 각 스토리보드는 독립적인 마크다운 파일로 작성됩니다.
- 모든 파일은 시스템 다이어그램을 포함해야 합니다.
- 오류 처리 시나리오를 반드시 포함해야 합니다.
