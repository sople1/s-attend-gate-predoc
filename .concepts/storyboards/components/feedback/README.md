# 피드백 패턴

이 디렉토리는 s-attend-gate의 사용자 피드백 패턴에 대한 스토리보드를 포함합니다.

## 피드백 유형

### 시각적 피드백
- 상태 표시
- 진행률 표시
- 성공/실패 표시
- 애니메이션

### 촉각적 피드백
- 진동 패턴
- 강도 변화
- 지속 시간

### 청각적 피드백
- 알림음
- 효과음
- 음성 안내

## 작성 지침

### 즉시성
- 실시간 응답
- 지연 최소화
- 상태 변화 명확화

### 의미 전달
- 직관적 이해
- 문맥 적절성
- 일관된 패턴

### 접근성 고려
- 다중 감각 피드백
- 사용자 설정 옵션
- 환경 적응성
