# User App

S-Attend-Gate의 모바일 사용자 애플리케이션입니다.

## 기능

- BLE 비콘 자동 감지
- GPS 위치 기반 검증
- QR 코드 백업 시스템
- 실시간 출석 상태 확인

## 기술 스택

- Framework: React Native
- State Management: Redux Toolkit
- BLE: react-native-ble-plx
- Location: react-native-geolocation
- QR Scanner: react-native-qrcode-scanner
