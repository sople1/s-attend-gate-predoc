# Integrated Platform

S-Attend-Gate의 통합 플랫폼 대시보드입니다.

## 기능

- 통합 대시보드
- 데이터 분석 및 시각화
- 시스템 모니터링
- 알림 관리
- 리포트 생성

## 기술 스택

- Framework: Next.js
- UI: Tailwind CSS
- Charts: D3.js
- State Management: Redux Toolkit
- API: GraphQL (Apollo Client)
- Real-time: WebSocket
