# Event Management

S-Attend-Gate의 이벤트 관리 백엔드 서버입니다.

## 기능

- 이벤트 생성 및 관리
- 참가자 관리
- 출석 데이터 처리
- 실시간 모니터링
- 보고서 생성

## 기술 스택

- Runtime: Node.js
- Framework: NestJS
- Database: PostgreSQL
- Cache: Redis
- Message Queue: RabbitMQ
- API: GraphQL, REST
